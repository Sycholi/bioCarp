# 用户习惯参考记录

本文件只记录可能有参考价值的使用习惯。用户水平和偏好不是固定事实。每次任务开始时都要重新判断当前使用者、当前任务和当前表达方式。

## 当前观察到的偏好

- 说明文件和输出图片默认使用 UTF-8 中文命名。
- 说明文件内容默认使用中文。
- 原始文件和关键输出文件必须保留。
- 删除非必要缓存前必须询问。
- 缺少分析工具或环境时，在不影响必要系统连接的前提下默认配置。
- 需要时使用多线程、GPU 和 sub-agent。
- 输出必须为最终可用结果。
